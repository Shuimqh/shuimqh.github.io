# 函数预编译
1、创建AO对象
2、将形参名和变量名作为AO对象属性名，属性值为undefined
3、实参形参相统一
4、将函数名作为AO对象属性名，属性值为函数体
# 案例1
~~~js
function fn(a) {
    console.log(a); // function a() {}
    var a = 123;
    console.log(a); // 123
    function a() {}
    console.log(a); // 123
    var b = function() {}
    console.log(b); // function() {}
    function d() {}
}
fn(1)
~~~
